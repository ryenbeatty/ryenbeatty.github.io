# Soehne (Klim Type Foundry)

Place your licensed Soehne web font files here. Required for the three-weight system:

| File | Weight | Use |
|------|--------|-----|
| `soehne-leicht.woff2` (optional: `soehne-leicht.woff`) | 300 | Body text, grid hint text |
| `soehne-buch.woff2` (optional: `soehne-buch.woff`) | 400 | Navigation, labels, captions, UI |
| `soehne-kraeftig.woff2` (optional: `soehne-kraeftig.woff`) | 500 | Headlines, site name |

If your export uses different filenames, either rename to match or update the `@font-face` `src` URLs in `src/styles/typography.css`.

[Soehne at Klim](https://klim.co.nz/collections/soehne/)
